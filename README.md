Projet Amazonie 2026 – Site Web

Ce site web accompagne le projet documentaire Amazonie 2026. Il propose des articles, des analyses et des contenus multimédias qui explorent les enjeux environnementaux, économiques et sociaux de l’Amazonie. L’ambition est de sensibiliser le public grâce à une approche à la fois moderne, immersive et résolument visuelle, inspirée de l’esthétique des magazines contemporains.
